<h1>영화예매 및 영화 굿즈 판매 홈페이지, GIGABOX</h1>
<p>제작기간 : 2024.11.25 ~ 2025.01.06</p>
<p>팀원 : 총 2명</p>
<br>

<h3>프로젝트 담당기능(프론트엔드 & 백엔드)</h3>
<ul>
  <li>아이디, 비밀번호 찾기</li>
  <li>영화예매, 좌석예매</li>
  <li>극장 페이지</li>

  <li>Spring boot3</li>
  <li>Java</li>
  <li>MySQL</li>

  <li>QueryDSL</li>
